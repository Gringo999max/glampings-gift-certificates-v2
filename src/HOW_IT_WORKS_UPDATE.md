# Обновление блока "Как это работает"

## Что изменилось

Полностью переработан блок "Как это работает" с использованием новых иллюстраций лисы.

## Изменённые компоненты

### 1. HowItWorksSection.tsx (обновлён)
- **Расположение:** `/components/HowItWorksSection.tsx`
- **Описание:** Основной компонент блока на главной странице
- **Дизайн:** Вариант 1 - горизонтальные карточки с большими иллюстрациями
- **Особенности:**
  - Три карточки в ряд на десктопе, в колонку на мобильных
  - Нумерованные бейджи на каждой карточке
  - Стрелки между шагами (только на десктопе)
  - Анимации при наведении и появлении
  - Кнопка CTA "Выбрать сертификат"

### 2. HeroSection.tsx (обновлено модальное окно)
- **Расположение:** `/components/HeroSection.tsx`
- **Описание:** Модальное окно при клике на кнопку "КАК ЭТО РАБОТАЕТ?"
- **Изменения:**
  - Заменено старое изображение на три карточки с иллюстрациями
  - Увеличена ширина модального окна (max-w-5xl вместо max-w-3xl)
  - Добавлены интерактивные карточки с hover-эффектами

### 3. HeroSection2.tsx (обновлено модальное окно)
- **Расположение:** `/components/HeroSection2.tsx`
- **Описание:** Альтернативная версия Hero секции
- **Изменения:** Идентичны HeroSection.tsx

### 4. App.tsx (обновлён импорт)
- **Изменение:** `NewHowItWorksSection` → `HowItWorksSection`

### 5. PetFriendlyGlampingPage.tsx (обновлён блок "Как это работает")
- **Расположение:** `/components/PetFriendlyGlampingPage.tsx`
- **Описание:** Блок "Как это работает" на странице глэмпинга с питомцем
- **Изменения:**
  - Заменены эмодзи-иконки на иллюстрации лисы (те же, что на главной)
  - Применён дизайн Варианта 1 с карточками
  - Фиолетовая цветовая схема (`bg-purple-600`, `border-purple-200`) вместо зелёной
  - Адаптированные тексты для отдыха с питомцем
  - Используются те же импорты `figma:asset` из основного компонента
