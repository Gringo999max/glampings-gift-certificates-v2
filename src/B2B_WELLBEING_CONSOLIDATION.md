## Текущая структура

### Корпоративная страница (`CorporateB2BPage.tsx`)
Полнофункциональная B2B Well-being страница с:

**Секции:**
1. Hero с Вариантом 3 сертификата (с аватарами команды)
2. Проблемы HR и решения
3. Преимущества Well-being программы
4. "Как это работает для HR" с лисичками
5. Выбор корпоративного Well-being пакета
6. Уже доверяют (социальное доказательство)
7. CTA секция
8. Форма заявки на корпоративные условия

**Особенности:**
- Полная валидация формы с react-hook-form
- Форматирование телефона
- Анимации Motion
- Адаптивный дизайн
- Footer с полной навигацией

## Доступ к странице

### Для пользователей:
- Через основное меню сайта → "Корпоративные подарки"
- Навигация вызовет `onNavigateToCorporate()` → маршрут `'corporate'`

### Для разработчиков:
- Плавающая кнопка с тестовыми страницами (оранжевая иконка справа снизу)
- Кнопка "B2B Well-being страница" в меню
