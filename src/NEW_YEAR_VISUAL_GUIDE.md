## 🚀 Быстрая установка

### 1️⃣ Только промо-блок (минимум)
```tsx
import { NewYearPromoVariant2 } from './NewYearPromoVariants';

<NewYearPromoVariant2 />
```

### 2️⃣ Промо + лёгкие декорации
```tsx
import { NewYearPromoVariant2 } from './NewYearPromoVariants';
import { FallingSnowflakes } from './NewYearDecorations';

<FallingSnowflakes count={20} speed="slow" />
<NewYearPromoVariant2 />
```

### 3️⃣ Полное новогоднее оформление
```tsx
import { NewYearPromoVariant1 } from './NewYearPromoVariants';
import { NewYearDecorationsPreset } from './NewYearDecorations';

<NewYearDecorationsPreset 
  includeSnow={true}
  includeGarland={true}
```
