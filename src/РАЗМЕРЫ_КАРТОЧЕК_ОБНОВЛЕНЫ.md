# ✅ РАЗМЕРЫ КАРТОЧЕК УНИФИЦИРОВАНЫ

## 🎯 Что сделано

Все табы в блоке **"Дарите как хотите!"** теперь используют **единую сетку** для консистентного размера карточек.

---

## 📊 БЫЛО vs СТАЛО

### ❌ БЫЛО (разные размеры):

**Электронный сертификат:**
```
Десктоп: 3 колонки → карточки 33.33% ширины = БОЛЬШИЕ ✓
┌──────────┬──────────┬──────────┐
│ Сертиф.1 │ Сертиф.2 │ Сертиф.3 │
├──────────┴──────────┴──────────┤
│       Сертификат 4             │
└────────────────────────────────┘
```

**Фирменный конверт:**
```
Десктоп: 4 колонки → карточки 25% ширины = МАЛЕНЬКИЕ ✗
┌─────┬─────┬─────┬─────┐
│Конв1│Конв2│Конв3│Конв4│
└─────┴─────┴─────┴─────┘
```

**Подарочная упаковка:**
```
Десктоп: 4 колонки → карточки 25% ширины = МАЛЕНЬКИЕ ✗
┌─────┬─────┬─────┬─────┐
│Кор.1│Кор.2│Кор.3│Кор.4│
└─────┴─────┴─────┴─────┘
```

**Проблема:** Карточки в конвертах/коробках на **33% меньше**, чем в сертификатах!

---

### ✅ СТАЛО (одинаковые размеры):

**ВСЕ ТАБЫ:**
```
Десктоп: 3 колонки → карточки 33.33% ширины = ОДИНАКОВЫЕ ✓
┌──────────┬──────────┬──────────┐
│ Элемент1 │ Элемент2 │ Элемент3 │
├──────────┴──────────┴──────────┤
│       Элемент 4                │
└────────────────────────────────┘
```

## 🎨 Единая сетка для всех табов

### Мобильные (< 768px):
- **1 колонка** - карточки занимают 100% ширины

### Планшеты (768-1024px):
- **2 колонки** - карточки по 50% ширины каждая

### Десктоп (> 1024px):
- **3 колонки** - карточки по 33.33% ширины каждая
- 4 элемента = 2 ряда (3 + 1)

### Gap (отступы):
- **8** - единообразно для всех табов

## 🔧 Технические изменения

### Файл: `/components/GiftOptionsSection.tsx`

**Функция `getGridClass` БЫЛО:**
```tsx
const getGridClass = useMemo(() => {
  switch (activeTab) {
    case 'electronic':
      return 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8'
    case 'envelope':
      return 'grid-cols-2 md:grid-cols-2 lg:grid-cols-4 gap-6'  // ← 4 колонки
    case 'gift':
      return 'grid-cols-2 md:grid-cols-2 lg:grid-cols-4 gap-6'  // ← 4 колонки
    case 'custom':
      return 'grid-cols-1 md:grid-cols-1 lg:grid-cols-1'
    default:
      return 'grid-cols-2 md:grid-cols-3 lg:grid-cols-3'
  }
}, [activeTab])
```

**Функция `getGridClass` СТАЛО:**
```tsx
const getGridClass = useMemo(() => {
  switch (activeTab) {
    case 'electronic':
      return 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8'
    case 'envelope':
      return 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8'  // ✓ 3 колонки
    case 'gift':
      return 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8'  // ✓ 3 колонки
    case 'custom':
      return 'grid-cols-1 md:grid-cols-1 lg:grid-cols-1'
    case 'default':
      return 'grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-8'  // ✓ 3 колонки
  }
}, [activeTab])
```
